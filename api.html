📚 API Documentation

Quick Start Guide

Authentication

curl -H "Authorization: Bearer YOUR_API_KEY" \\
    https://api.gateways.com/v1/proxies

Include your API key in the Authorization header for all requests.

Get Proxy List

GET /v1/proxies?type=residential&country=US&limit=10

Retrieve available proxies with optional filtering parameters.

Rotate Proxy

POST /v1/proxies/rotate
{
  "proxy_id": "px_12345",
  "session_id": "sess_67890"
}

Force rotation of a specific proxy session.
